    CompileTask

    Extends:

    src/lib/Task~Task → CompileTask

    Compiles code from one language to another using a Compiler. May compile in three ways:

    • Compiling a chunk (string) to another chunk.
    • Compiling a file from a source path to a target path.
    • Compiling a directory of files recursively from a source path to a target path, compiling all files that match.
