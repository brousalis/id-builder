    PlantUMLCompile

    You can directly use instance of this class. plantUMLCompile

    Extends:

    src/lib/CompileTask~CompileTask → PlantUMLCompile

    Constructs a PlantUML version of all classes fed to the compiler and puts them into one diagram. Only supports a very specific subset of ES6 functionality;

    • Every module included in the graph contains one class.
    • Class names are unique (for now).
